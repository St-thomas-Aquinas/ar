Interactive 3D Human Body Anatomy

Web & Virtual Reality (VR) Educational Project

About This Project

This project demonstrates the human body anatomy using a 3D interactive model. It is created using modern web technologies and supports Virtual Reality (VR), allowing students to explore the human body in an immersive way.

How to Use

  • Rotate the model using your mouse or touch
  • Zoom in and out to view organs closely
  • Click Enter VR to experience in Virtual Reality
  • Works on mobile, laptop, and VR headsets
